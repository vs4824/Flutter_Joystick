# Flutter Joystick

A virtual joystick for Flutter applications.

## Joystick

   ```
   Joystick(listener: (details) {
...
})
   ```

## Joystick Area 

JoystickArea allows to render a joystick anywhere in this area where user clicks.

   ```
   JoystickArea(
  listener: (details) {
    ...
  },
  child: ...
)
   ```
